# Typography Pairings Reference

## Format
Each pairing includes: Heading Font, Body Font, Mood/Keywords, Best For, CSS Import.

## Professional & Corporate

### Modern Professional
- **Fonts:** Poppins (heading) + Open Sans (body)
- **Mood:** modern, professional, clean, corporate, friendly
- **Best For:** SaaS, corporate sites, business apps, startups

## Typography Rules

### Size Scale (rem)
- **Display:** 3-4rem (48-64px)
- **H1:** 2-2.5rem (32-40px)
- **H2:** 1.5-1.75rem (24-28px)
- **H3:** 1.25rem (20px)
- **Body:** 1rem (16px)
- **Small:** 0.875rem (14px)

### Line Height
- **Headings:** 1.1-1.3
- **Body:** 1.5-1.75
- **UI/Labels:** 1.2-1.4

### Line Length
- **Optimal:** 65-75 characters
- **Max:** 80 characters
- **Min:** 45 characters
