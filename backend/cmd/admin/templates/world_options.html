SMTP (outgoing email)

Used for verification and password-reset emails. In dev, point at MailHog (host=mailhog, port=1025).

Public app URL

Base URL used in email templates (verification, password reset). Override if behind a reverse proxy on 443.

YouTube live streaming

Default RTMP target for the "Stream to YouTube" recording action. Leave empty to disable YouTube streaming (MP4 still works). The host can override per-recording via the confirm modal.

ffmpeg audio extraction

Limits applied to ffmpeg when extracting MP3 audio from recorded MP4 meetings. Hot-reloaded on save.

World king

Display-only. The king's name is shown on the welcome page footer; the email is visible only here and is the default recipient for error emails when mode = "king".

Error email notifications

The audit service emails these recipients when an audit event with severity "error" is emitted (e.g. recording.audio_extraction_failed). Hot-reloaded on save.

Meeting recording

When disabled, ext-rec refuses recording.start requests and the frontend disables the Record button. Hot-reloaded on save.

Read-only (set via env var)

These are read at container startup and cannot be hot-reloaded. Change them in docker-compose.yml / .env and restart the stack.
